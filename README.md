# Novel Helper

自用插件，使用可自行打包

VSCode小说创作辅助插件，提供目录管理、文本高亮、字数统计、排版格式化等功能，助力小说创作效率提升。

## 功能特性

### 📁 标准化目录管理
- 初始化一键创建小说创作标准目录结构（大纲、设定、素材、正文）
- 树视图可视化管理，支持新建总大纲、分大纲、角色设定、事物设定、分卷、章节等
- 自动隐藏插件配置文件，保持目录整洁

### 🔍 智能文本高亮
- 从设定文件中选中文本快速添加高亮项（角色名、事物名等）
- 高亮文本可点击直接跳转到设定源文件，方便快速查阅
- 自定义高亮颜色和文本颜色

### 📊 创作数据统计
- 实时统计当前文档字数
- 计算码字速度（字/分钟）
- 累计码字时间，支持格式化显示（时/分/秒）
- 提供统计信息查看命令

### ✍️ 排版辅助
- 自定义段首缩进空格数
- 自定义行间空行数
- 一键格式化文档，自动应用排版规则

### 📈 状态栏实时显示
- 状态栏显示字数、缩进数、空行数、码字速度、累计时间
- 点击状态栏项快速呼出配置面板

### ⚙️ 灵活配置面板
- 聚合所有可配置项，类似VSCode命令面板的交互方式
- 支持实时修改排版参数、高亮样式等

## 快速开始

### 1. 初始化工作区
1. 打开VSCode，新建并打开一个空文件夹
2. 打开命令面板（`Ctrl+Shift+P`/`Cmd+Shift+P`）
3. 执行命令：`Novel Helper: 初始化小说工作区`
4. 插件会自动创建标准目录结构和初始文件

### 2. 使用树视图管理内容
- 侧边栏「小说创作面板」中，每个目录下都有「新建XX」选项
- 点击「新建总大纲」可创建大纲文件
- 点击「新建角色设定」可创建角色设定文件
- 正文目录下可创建分卷和章节（txt文件）

### 3. 设置文本高亮
1. 打开设定文件（如「设定/角色设定/主角.md」）
2. 选中角色名（如「张三」）
3. 在弹出的提示中选择「是」，将其添加为高亮项
4. 在正文文件中输入「张三」会自动高亮，点击可跳转到设定文件

### 4. 排版格式化
1. 打开正文txt文件
2. 打开命令面板，执行`Novel Helper: 格式化文档`
3. 或在配置面板中修改「段首缩进」「行间空行」参数，自动应用

### 5. 查看统计信息
- 打开命令面板，执行`Novel Helper: 显示创作统计`
- 查看总字数、总耗时、平均码字速度
